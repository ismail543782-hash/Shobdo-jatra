# শব্দযাত্রা — সম্পূর্ণ প্রজেক্ট

এই ফোল্ডারে আপনার পুরো গেমের কোড আছে, যেটা এখন থেকে সম্পূর্ণভাবে আপনার নিয়ন্ত্রণে। নিচে ধাপে ধাপে সব বলা হলো।

---

## ১. প্রথমবার চালানো (নিজের কম্পিউটারে)

**যা লাগবে:** [Node.js](https://nodejs.org) ইনস্টল করা থাকতে হবে (LTS ভার্সন)।

```bash
# এই ফোল্ডারে ঢুকে
npm install
npm run dev
```

টার্মিনালে একটা লিংক দেখাবে (সাধারণত `http://localhost:5173`) — ব্রাউজারে খুললেই গেম চলবে।

---

## ২. আপনার নিয়ন্ত্রণে রাখা — GitHub-এ কোড রাখুন

এটা করলে কোড কখনো হারাবে না, এবং পরে হোস্টিং সহজ হবে।

1. [github.com](https://github.com) -এ ফ্রি অ্যাকাউন্ট খুলুন (না থাকলে)
2. একটা নতুন repository বানান (নাম দিন `shobdojatra`)
3. এই ফোল্ডারের কোড আপলোড করুন:
```bash
git init
git add .
git commit -m "প্রথম ভার্সন"
git branch -M main
git remote add origin https://github.com/আপনার-ইউজারনেম/shobdojatra.git
git push -u origin main
```

---

## ৩. ফ্রি-তে হোস্ট করা (সবাই যেন খেলতে পারে)

**Vercel** সবচেয়ে সহজ ও সম্পূর্ণ ফ্রি:

1. [vercel.com](https://vercel.com) -এ যান, GitHub দিয়ে সাইন আপ করুন
2. "Add New Project" → আপনার `shobdojatra` repository বেছে নিন
3. Vercel নিজে থেকেই বুঝে নেবে এটা Vite প্রজেক্ট — শুধু "Deploy" চাপুন
4. ২ মিনিটে একটা লিংক পাবেন (যেমন `shobdojatra.vercel.app`) যেটা দিয়ে যে কেউ, যেকোনো জায়গা থেকে খেলতে পারবে

কোড পরে বদলালে শুধু `git push` করলেই Vercel নিজে থেকে আপডেট হয়ে যাবে।

(Netlify -ও একইভাবে কাজ করে, চাইলে সেটাও ব্যবহার করতে পারেন।)

## ৪. কে কীভাবে ব্যবহার করছে — সেটা দেখা (অ্যানালিটিক্স)

এখন গেমটাতে কোনো ট্র্যাকিং নেই, তাই কেউ খেললে আপনি জানতে পারবেন না। এটা যোগ করতে:

- **সহজ পথ:** Vercel Analytics (ফ্রি টায়ারে সীমিত ব্যবহার আছে) — কতজন ভিজিট করলো তা দেখাবে
- **বিস্তারিত পথ:** [Google Analytics](https://analytics.google.com) (সম্পূর্ণ ফ্রি) — `index.html`-এ একটা ছোট স্ক্রিপ্ট যোগ করলেই ভিজিটর সংখ্যা, কোন দেশ থেকে আসছে, কতক্ষণ খেলছে ইত্যাদি দেখা যাবে

কে কোন স্কোর করলো, কে কতবার খেলল — এই ধরনের ব্যবহারকারী-ভিত্তিক তথ্য দেখতে হলে সত্যিকারের অ্যাকাউন্ট সিস্টেম (Firebase) লাগবে, যা নিয়ে আগেই বলেছিলাম।

## ৫. মোবাইল অ্যাপ বানিয়ে Play Store-এ দেওয়া

ওয়েব ভার্সনটা ভালোভাবে কাজ করলে, এটাকে Android অ্যাপে রূপান্তর করা যায় **Capacitor** দিয়ে (ফ্রি টুল, কোড প্রায় অপরিবর্তিত থাকে):

```bash
npm run build
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init shobdojatra com.yourname.shobdojatra
npx cap add android
npx cap sync
npx cap open android
```

এরপর Android Studio খুলে যাবে, সেখান থেকে `.aab` ফাইল তৈরি করে Google Play Console-এ (**একবার মাত্র $25 ফি**) আপলোড করলেই অ্যাপ পাবলিশ হয়ে যাবে।
